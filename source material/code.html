Code

Rhino was first developed using BASIC (Beginners All-purpose Symbolic Instruction Code) which was a common programming language for early microcomputers.

This version is created using:

  • HTML - HyperText Mark-up Language - used to arrange the words and diagrams
  • CSS - Cascading Style Sheets - used to set the style and colours and
  • Javascript - used to program the snooker ball, set values and navigate between pages
  • Bootstrap - used to make the pages pretty and responsive

The code can be found on the GitHub website.
